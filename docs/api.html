API Reference

This page provides a detailed reference for all available methods and properties in TaleemCanvas.

Creating a TaleemCanvas Instance

To start using TaleemCanvas, create an instance with a reference to an existing `` element and its 2D context:


const canvasElement = document.getElementById("myCanvas");
const ctx = canvasElement.getContext("2d");
const canvas = new TaleemCanvas(canvasElement, ctx);

Core Methods

  • canvas.draw() → Renders all items on the canvas.
  • canvas.start() → Starts the animation loop.
  • canvas.stop() → Stops the animation loop.
  • canvas.addItems(itemExtrasArray) → Adds an array of item configurations.
  • canvas.deleteItem(item) → Removes a specific item.
  • canvas.deleteAllItems() → Clears all items from the canvas.
  • canvas.cloneItem(item) → Creates a copy of an existing item.
  • canvas.save() → Returns an array of item configurations for saving.

Event Handling

TaleemCanvas allows users to listen to mouse and keyboard events on the canvas.


canvas.onMouse("click", (event, item) => {
    if (item) {
        console.log("Clicked on item:", item);
    }
});

canvas.onKey("keydown", (event) => {
    console.log("Key pressed:", event.key);
});

Adding Items

Use the `.add` method to create and add items dynamically.

  • canvas.add.text() → Adds a text item.
  • canvas.add.image() → Adds an image item.
  • canvas.add.sprite() → Adds a sprite item.
  • canvas.add.list() → Adds a list item.
  • canvas.add.piechart() → Adds a pie chart item.
  • canvas.add.circle() → Adds a circle item.
  • canvas.add.ellipse() → Adds an ellipse item.
  • canvas.add.line() → Adds a line item.
  • canvas.add.ray() → Adds a ray item.
  • canvas.add.rectangle() → Adds a rectangle item.
  • canvas.add.triangle() → Adds a triangle item.

Item Properties

Each item has customizable properties:

  • item.set("property", value) → Updates a property value.
  • item.x → Get or set the X position.
  • item.y → Get or set the Y position.
  • item.width → Get or set the width.
  • item.height → Get or set the height.
  • item.boundingRectangleX() → Get X coordinate of bounding box.
  • item.boundingRectangleY() → Get Y coordinate of bounding box.
  • item.boundingRectangleWidth() → Get bounding box width.
  • item.boundingRectangleHeight() → Get bounding box height.

Saving & Loading Items

You can save the current state of the canvas and restore it later:


const savedData = canvas.save();
canvas.addItems(savedData);
