API Reference

This page provides a detailed reference for all available methods in TaleemCanvas.

TaleemCanvas Methods

  • new TaleemCanvas(canvasId) → Creates a new canvas instance.
  • canvas.draw() → Renders all items on the canvas.
  • canvas.add.text() → Adds a text item.
  • canvas.add.image() → Adds an image item.
  • canvas.add.sprite() → Adds a sprite item.
  • canvas.add.list() → Adds a list item.
  • canvas.add.piechart() → Adds a pie chart item.
  • canvas.add.circle() → Adds a circle item.
  • canvas.add.ellipse() → Adds an ellipse item.
  • canvas.add.line() → Adds a line item.
  • canvas.add.ray() → Adds a ray item.
  • canvas.add.rectangle() → Adds a rectangle item.
  • canvas.add.triangle() → Adds a triangle item.

Common Item Methods

  • item.set("property", value) → Sets a property value.
  • item.x → Get or set the X position.
  • item.y → Get or set the Y position.
  • item.width → Get or set the width (if applicable).
  • item.height → Get or set the height (if applicable).
  • item.boundingRectangleX() → Get the X position of the bounding box.
  • item.boundingRectangleY() → Get the Y position of the bounding box.
  • item.boundingRectangleWidth() → Get the width of the bounding box.
  • item.boundingRectangleHeight() → Get the height of the bounding box.

Working with Sprites

Sprites allow selecting specific parts of an image using predefined coordinates.


const sprite = canvas.add.sprite();
sprite.set("src", "people");
sprite.setSelectedItem("mf_wlk_biz");

Loading Images from TaleemAssets (Optional)

If using taleem-assets, preload images before assigning them.


await assets.loadImages(["wood.jpg"]);
const imageData = assets.getImage("wood.jpg");
image.set("src", imageData.img.src);
