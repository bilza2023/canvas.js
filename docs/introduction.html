TaleemCanvas Documentation

Welcome to the official documentation for TaleemCanvas, a JavaScript library designed for creating slides specially educational slides on html5 canvas. TaleemCanvas allows users to use images, sprites etc with ease.

What is TaleemCanvas?

TaleemCanvas is built around the concept of drawable items. Users can add, delete, and modify different graphical elements to compose interactive presentations. At this time, the library provides basic drawing capabilities, including:

  • Shapes: Circle, Rectangle, Ellipse, Triangle, Line, Ray
  • Text
  • Images
  • Lists
  • Pie Charts
  • Sprites

Integration with Taleem Assets

TaleemCanvas is designed to work with Taleem Assets, a bundle that provides icons, background images, sprites, and other assets. While TaleemCanvas can be used independently, features like images, sprites, and background images will not function without Taleem Assets.

Available Items

  • Text Item
  • Image Item
  • Sprite Item
  • List Item
  • Pie Chart Item
  • Circle Item
  • Ellipse Item
  • Line Item
  • Ray Item
  • Rectangle Item
  • Triangle Item

Controlling the Canvas

TaleemCanvas provides event-handling capabilities to allow user interaction. For example, you can listen for mouse events using the following method:


    onMouse(eventType, callback) {
        this.eventModule.on(eventType, callback);
    }

For more details, visit the Event Handling page.
