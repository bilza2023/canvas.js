Getting Started with TaleemCanvas

TaleemCanvas is a lightweight JavaScript library for drawing and managing canvas-based elements.

Installation

Install the package using npm:


npm install taleem-canvas

Basic Usage

Here’s how to create a canvas and add basic elements:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");

const text = canvas.add.text();
text.set("text", "Hello, Taleem!");
text.set("fontSize", 24);
text.x = 100;
text.y = 50;

const rectangle = canvas.add.rectangle();
rectangle.set("width", 150);
rectangle.set("height", 100);
rectangle.set("fillColor", "blue");
rectangle.x = 200;
rectangle.y = 150;

canvas.draw();

Canvas Initialization

Create a `` element in your HTML and provide its ID to TaleemCanvas.


<canvas id="myCanvas" width="800" height="600"></canvas>

Adding Items to Canvas

You can add various items to the canvas using the `.add` method:


const text = canvas.add.text();
const image = canvas.add.image();
const sprite = canvas.add.sprite();
const list = canvas.add.list();
const pieChart = canvas.add.piechart();
const circle = canvas.add.circle();
const ellipse = canvas.add.ellipse();
const line = canvas.add.line();
const ray = canvas.add.ray();
const rectangle = canvas.add.rectangle();
const triangle = canvas.add.triangle();

Drawing the Canvas

Once you’ve added all items, call canvas.draw() to render them.


canvas.draw();
