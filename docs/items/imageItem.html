Image Item

The Image Item allows you to render images on the canvas. You can set the source, position, and size.

Example Usage

This example demonstrates how to add an image item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const image = canvas.add.image();

image.set("src", "wood.jpg");
image.set("globalAlpha", 1); // Full opacity

// Set position and size
image.x = 50;
image.y = 50;
image.width = 200;
image.height = 200;

canvas.draw();

Preview

Available Methods

  • set("src", "image.jpg") → Set the image source.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.
  • width → Get or set image width.
  • height → Get or set image height.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Image Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
