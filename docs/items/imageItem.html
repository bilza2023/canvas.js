Image Item

The ImageItem class is used to display images within TaleemCanvas. Images are referenced by their source file name and loaded from the provided assets.

Usage

// Creating an Image Item
const image = canvas.add.image();
image.set("src", "example.jpg");
image.x = 100;
image.y = 100;
image.width = 200;
image.height = 200;

Available Properties

  • x - X-coordinate of the image.
  • y - Y-coordinate of the image.
  • width - Width of the image.
  • height - Height of the image.
  • src - Source filename of the image (must be available in assets).
  • globalAlpha - Opacity level of the image (0 to 1).

Example

Rendering

The draw method attempts to retrieve the image from the asset manager. If the image is missing, a placeholder gray rectangle with an error message is displayed.

Methods

  • set(key, value) - Updates the given property.
  • boundingRectangleX() - Returns the bounding X position.
  • boundingRectangleY() - Returns the bounding Y position.
  • boundingRectangleWidth() - Returns the bounding width.
  • boundingRectangleHeight() - Returns the bounding height.
