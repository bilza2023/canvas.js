Circle Item

The Circle item in TaleemCanvas allows users to draw circular shapes with customizable properties such as position, radius, stroke width, and color.

Creating a Circle


const circle = canvas.add.circle();
circle.x = 150;
circle.y = 150;
circle.set("radius", 50);
circle.set("color", "gray");
circle.set("lineWidth", 2);
circle.set("filled", true);
canvas.draw();

Circle Properties

  • circle.x → Get or set the X position.
  • circle.y → Get or set the Y position.
  • circle.set("radius", value) → Set the radius of the circle.
  • circle.set("color", value) → Set the stroke or fill color.
  • circle.set("lineWidth", value) → Set the stroke width.
  • circle.set("filled", true/false) → Determines if the circle is filled or just an outline.
  • circle.globalAlpha → Adjust the transparency level.

Bounding Box

The bounding box of a circle is a square calculated based on its radius:


const width = circle.boundingRectangleWidth();
const height = circle.boundingRectangleHeight();

Example

Below is an example of adding a filled red circle to the canvas:


const circle = canvas.add.circle();
circle.x = 200;
circle.y = 200;
circle.set("radius", 75);
circle.set("color", "red");
circle.set("filled", true);
canvas.draw();
