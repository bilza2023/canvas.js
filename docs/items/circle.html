Circle Item

The Circle Item allows you to draw a circle on the canvas with customizable size, color, and outline.

Example Usage

This example demonstrates how to add a circle item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const circle = canvas.add.circle();

circle.set("radius", 50);
circle.set("fillColor", "blue");
circle.set("strokeColor", "black");
circle.set("strokeWidth", 2);
circle.set("globalAlpha", 1); // Full opacity

// Set position
circle.x = 200;
circle.y = 200;

canvas.draw();

Preview

Available Methods

  • set("radius", 50) → Set the radius of the circle.
  • set("fillColor", "blue") → Set the fill color.
  • set("strokeColor", "black") → Set the stroke (outline) color.
  • set("strokeWidth", 2) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Circle Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
