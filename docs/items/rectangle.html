Rectangle Item

The Rectangle Item allows you to draw a rectangle on the canvas with customizable size, color, and outline.

Example Usage

This example demonstrates how to add a rectangle item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const rectangle = canvas.add.rectangle();

rectangle.set("width", 150);
rectangle.set("height", 100);
rectangle.set("fillColor", "orange");
rectangle.set("strokeColor", "black");
rectangle.set("strokeWidth", 2);
rectangle.set("globalAlpha", 1); // Full opacity

// Set position
rectangle.x = 200;
rectangle.y = 150;

canvas.draw();

Preview

Available Methods

  • set("fillColor", "orange") → Set the fill color.
  • set("strokeColor", "black") → Set the stroke (outline) color.
  • set("strokeWidth", 2) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.
  • width → Get or set rectangle width.
  • height → Get or set rectangle height.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Rectangle Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
