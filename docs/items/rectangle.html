Rectangle Item

The Rectangle item in TaleemCanvas allows users to draw rectangular shapes with customizable properties such as position, size, stroke width, and color.

Creating a Rectangle


const rectangle = canvas.add.rectangle();
rectangle.x = 100;
rectangle.y = 100;
rectangle.set("width", 150);
rectangle.set("height", 100);
rectangle.set("color", "blue");
rectangle.set("lineWidth", 2);
rectangle.set("filled", true);
canvas.draw();

Rectangle Properties

  • rectangle.x → Get or set the X position.
  • rectangle.y → Get or set the Y position.
  • rectangle.set("width", value) → Set the width of the rectangle.
  • rectangle.set("height", value) → Set the height of the rectangle.
  • rectangle.set("color", value) → Set the stroke or fill color.
  • rectangle.set("lineWidth", value) → Set the stroke width.
  • rectangle.set("filled", true/false) → Determines if the rectangle is filled or just an outline.
  • rectangle.globalAlpha → Adjust the transparency level.

Bounding Box

The bounding box of a rectangle is determined by its width and height:


const width = rectangle.boundingRectangleWidth();
const height = rectangle.boundingRectangleHeight();

Example

Below is an example of adding a filled green rectangle to the canvas:


const rectangle = canvas.add.rectangle();
rectangle.x = 250;
rectangle.y = 200;
rectangle.set("width", 200);
rectangle.set("height", 120);
rectangle.set("color", "green");
rectangle.set("filled", true);
canvas.draw();
