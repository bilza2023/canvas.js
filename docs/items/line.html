Line Item

The Line class is used to draw straight lines between two points.

Usage

// Creating a Line Item
const line = canvas.add.line();
line.x1 = 50;
line.y1 = 50;
line.x2 = 200;
line.y2 = 200;
line.set("color", "blue");
line.set("lineWidth", 2);

Available Properties

  • x1 - X-coordinate of the start point.
  • y1 - Y-coordinate of the start point.
  • x2 - X-coordinate of the end point.
  • y2 - Y-coordinate of the end point.
  • lineWidth - Thickness of the line.
  • color - Color of the line.
  • globalAlpha - Opacity level of the line (0 to 1).

Example

Methods

  • set(key, value) - Updates the given property.
  • boundingRectangleX() - Returns the bounding X position.
  • boundingRectangleY() - Returns the bounding Y position.
  • boundingRectangleWidth() - Returns the bounding width.
  • boundingRectangleHeight() - Returns the bounding height.
