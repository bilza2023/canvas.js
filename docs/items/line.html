Line Item

The Line Item allows you to draw a straight line on the canvas between two points.

Example Usage

This example demonstrates how to add a line item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const line = canvas.add.line();

line.set("x2", 300);  // X coordinate of the end point
line.set("y2", 200);  // Y coordinate of the end point
line.set("strokeColor", "red");
line.set("strokeWidth", 3);
line.set("globalAlpha", 1); // Full opacity

// Set starting position
line.x = 100;
line.y = 100;

canvas.draw();

Preview

Available Methods

  • set("x2", 300) → Set the X coordinate of the end point.
  • set("y2", 200) → Set the Y coordinate of the end point.
  • set("strokeColor", "red") → Set the stroke (line) color.
  • set("strokeWidth", 3) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set the starting X position.
  • y → Get or set the starting Y position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Line Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
