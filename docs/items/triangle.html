Triangle Item

The Triangle Item allows you to draw a triangle on the canvas with customizable size, color, and outline.

Example Usage

This example demonstrates how to add a triangle item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const triangle = canvas.add.triangle();

triangle.set("x1", 100);
triangle.set("y1", 100);
triangle.set("x2", 50);
triangle.set("y2", 200);
triangle.set("x3", 200);
triangle.set("y3", 200);
triangle.set("color", "red");
triangle.set("filled", false);
triangle.set("lineWidth", 2);
triangle.set("globalAlpha", 1); // Full opacity

canvas.draw();

Preview

Available Methods

  • set("color", "red") → Set the fill or outline color.
  • set("filled", true/false) → Determines if the triangle is filled or just an outline.
  • set("lineWidth", 2) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x1, y1 → Get or set the first vertex position.
  • x2, y2 → Get or set the second vertex position.
  • x3, y3 → Get or set the third vertex position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Triangle Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
