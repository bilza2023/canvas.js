List Item

The List class is used to display a list of text items with customizable properties.

Usage

// Creating a List Item
const list = canvas.add.list();
list.x = 100;
list.y = 100;
list.set("listArray", ["Item 1", "Item 2", "Item 3"]);
list.set("fontSize", 20);
list.set("color", "black");

Available Properties

  • x - X-coordinate of the list.
  • y - Y-coordinate of the list.
  • listArray - Array of list items.
  • fontSize - Size of the text.
  • fontFamily - Font family used.
  • color - Color of the text.
  • lineGap - Spacing between list items.
  • indentation - Indentation for nested items.
  • globalAlpha - Opacity level of the list (0 to 1).

Example

Methods

  • set(key, value) - Updates the given property.
  • boundingRectangleX() - Returns the bounding X position.
  • boundingRectangleY() - Returns the bounding Y position.
  • boundingRectangleWidth() - Returns the bounding width.
  • boundingRectangleHeight() - Returns the bounding height.
