List Item

The List Item allows you to display a list of text items on the canvas with indentation and line spacing.

Example Usage

This example demonstrates how to add a list item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const list = canvas.add.list();

list.set("listArray", [
    "First item",
    "Second item",
    "Third item"
]);
list.set("fontSize", 20);
list.set("fontFamily", "Arial");
list.set("color", "black");
list.set("lineGap", 5);
list.set("indentation", 5);

// Set position
list.x = 100;
list.y = 100;

canvas.draw();

Preview

Available Methods

  • set("listArray", [...]) → Set the array of text items.
  • set("fontSize", 20) → Set the font size.
  • set("fontFamily", "Arial") → Set the font family.
  • set("color", "black") → Set the text color.
  • set("lineGap", 5) → Set the gap between lines.
  • set("indentation", 5) → Set indentation for each new line.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.
