Sprite Item

The Sprite Item allows you to render a part of a sprite sheet on the canvas. It uses taleem-assets to fetch the sprite image and item data.

Example Usage

This example demonstrates how to add a sprite item to the canvas:


import TaleemCanvas from "taleem-canvas";
import Assets from "taleem-assets";

const assets = new Assets();
const canvas = new TaleemCanvas("myCanvas", assets);
const sprite = canvas.add.sprite();

// Set the sprite sheet source
sprite.set("src", "people"); // Must match an available sprite in taleem-assets

// Get available sprite items
const availableItems = sprite.getAvailableItems();
if (availableItems.length > 0) {
    sprite.setSelectedItem(availableItems[0]); // Select the first item by default
}

canvas.draw();

Preview

Available Methods

  • set("src", "people") → Set the sprite sheet source.
  • getAvailableItems() → Get a list of available items in the sprite.
  • setSelectedItem("itemName") → Select a specific sprite item.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.
  • width → Get or set sprite display width.
  • height → Get or set sprite display height.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Only With Assets

The Sprite Item requires taleem-assets to function. It fetches the sprite sheet and item data from assets.
