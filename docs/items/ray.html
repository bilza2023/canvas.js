Ray Item

The Ray Item allows you to draw a ray that starts at a point and extends infinitely in a specified direction.

Example Usage

This example demonstrates how to add a ray item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const ray = canvas.add.ray();

ray.set("angle", 45);  // Angle of the ray in degrees
ray.set("strokeColor", "green");
ray.set("strokeWidth", 3);
ray.set("globalAlpha", 1); // Full opacity

// Set starting position
ray.x = 150;
ray.y = 150;

canvas.draw();

Preview

Available Methods

  • set("angle", 45) → Set the angle of the ray in degrees.
  • set("strokeColor", "green") → Set the stroke (line) color.
  • set("strokeWidth", 3) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set the starting X position.
  • y → Get or set the starting Y position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Ray Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
