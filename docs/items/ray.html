Ray Item

The Ray class is used to draw a directional line with optional arrowheads.

Usage

// Creating a Ray Item
const ray = canvas.add.ray();
ray.x1 = 150;
ray.y1 = 150;
ray.x2 = 350;
ray.y2 = 250;
ray.set("color", "red");
ray.set("lineWidth", 2);
ray.set("startArrow", true);
ray.set("endArrow", true);

Available Properties

  • x1 - X-coordinate of the starting point.
  • y1 - Y-coordinate of the starting point.
  • x2 - X-coordinate of the ending point.
  • y2 - Y-coordinate of the ending point.
  • arrowWidth - Width of the arrowhead.
  • arrowHeight - Height of the arrowhead.
  • startArrow - Boolean to show/hide the start arrow.
  • endArrow - Boolean to show/hide the end arrow.
  • lineWidth - Thickness of the ray line.
  • color - Color of the ray line.
  • globalAlpha - Opacity level of the ray (0 to 1).
