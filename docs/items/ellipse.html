Ellipse Item

The Ellipse item in TaleemCanvas allows users to draw elliptical shapes with customizable properties such as position, radius, stroke width, and color.

Creating an Ellipse


const ellipse = canvas.add.ellipse();
ellipse.x = 200;
ellipse.y = 200;
ellipse.set("radiusX", 60);
ellipse.set("radiusY", 40);
ellipse.set("color", "red");
ellipse.set("lineWidth", 2);
ellipse.set("filled", true);
canvas.draw();

Ellipse Properties

  • ellipse.x → Get or set the X position.
  • ellipse.y → Get or set the Y position.
  • ellipse.set("radiusX", value) → Set the horizontal radius.
  • ellipse.set("radiusY", value) → Set the vertical radius.
  • ellipse.set("color", value) → Set the stroke or fill color.
  • ellipse.set("lineWidth", value) → Set the stroke width.
  • ellipse.set("filled", true/false) → Determines if the ellipse is filled or just an outline.
  • ellipse.globalAlpha → Adjust the transparency level.

Bounding Box

The bounding box of an ellipse is determined by its radii:


const width = ellipse.boundingRectangleWidth();
const height = ellipse.boundingRectangleHeight();

Example

Below is an example of adding a filled yellow ellipse to the canvas:


const ellipse = canvas.add.ellipse();
ellipse.x = 300;
ellipse.y = 250;
ellipse.set("radiusX", 80);
ellipse.set("radiusY", 50);
ellipse.set("color", "yellow");
ellipse.set("filled", true);
canvas.draw();
