Ellipse Item

The Ellipse Item allows you to draw an ellipse on the canvas with customizable size, color, and outline.

Example Usage

This example demonstrates how to add an ellipse item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const ellipse = canvas.add.ellipse();

ellipse.set("radiusX", 80);
ellipse.set("radiusY", 50);
ellipse.set("fillColor", "purple");
ellipse.set("strokeColor", "black");
ellipse.set("strokeWidth", 2);
ellipse.set("globalAlpha", 1); // Full opacity

// Set position
ellipse.x = 250;
ellipse.y = 200;

canvas.draw();

Preview

Available Methods

  • set("radiusX", 80) → Set the horizontal radius of the ellipse.
  • set("radiusY", 50) → Set the vertical radius of the ellipse.
  • set("fillColor", "purple") → Set the fill color.
  • set("strokeColor", "black") → Set the stroke (outline) color.
  • set("strokeWidth", 2) → Set the width of the stroke.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Ellipse Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
