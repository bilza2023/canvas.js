Text Item

The Text Item allows you to render text on the canvas. You can set its position, color, font size, and font family.

Example Usage

This example demonstrates how to add a text item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas"); // ✅ Works without taleem-assets
const text = canvas.add.text();

text.set("text", "Hello, Taleem!");
text.set("fontSize", 24);
text.set("color", "white");
text.set("fontFamily", "Arial");
text.set("globalAlpha", 1); // Full opacity

// Set position
text.x = 100;
text.y = 50;

canvas.draw();

Preview

Available Methods

  • set("text", "Your text here") → Set the text content.
  • set("fontSize", 24) → Set font size.
  • set("color", "red") → Set text color.
  • set("fontFamily", "Arial") → Set font family.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set horizontal position.
  • y → Get or set vertical position.
  • width → Get calculated text width (based on content and font size).
  • height → Get calculated text height (measured using the letter "W").

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the calculated width of the text.
  • boundingRectangleHeight() → Get the calculated height of the text.

Works Without Assets

The Text Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
