Pie Chart Item

The PieChart class is used to display a pie chart with multiple data segments.

Usage

// Creating a Pie Chart Item
const piechart = canvas.add.piechart();
piechart.x = 200;
piechart.y = 200;
piechart.set("radius", 100);
piechart.set("data", [
    { label: "A", percentage: 30, color: "red" },
    { label: "B", percentage: 50, color: "blue" },
    { label: "C", percentage: 20, color: "green" }
]);
piechart.set("showLabels", true);
piechart.set("labelFontSize", 14);
piechart.set("labelColor", "black");

Available Properties

  • x - X-coordinate of the pie chart center.
  • y - Y-coordinate of the pie chart center.
  • radius - Radius of the pie chart.
  • data - Array of data segments (label, percentage, color).
  • showLabels - Whether to display labels on the chart.
  • labelFontSize - Font size for labels.
  • labelColor - Color of the labels.
  • globalAlpha - Opacity level of the pie chart (0 to 1).
