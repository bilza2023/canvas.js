Pie Chart Item

The Pie Chart Item allows you to render a pie chart on the canvas, displaying multiple data segments with customizable labels and colors.

Example Usage

This example demonstrates how to add a pie chart item to the canvas:


import TaleemCanvas from "taleem-canvas";

const canvas = new TaleemCanvas("myCanvas");
const pieChart = canvas.add.piechart();

pieChart.set("x", 300);
pieChart.set("y", 200);
pieChart.set("radius", 100);
pieChart.set("data", [
    { label: "A", percentage: 30, color: "red" },
    { label: "B", percentage: 50, color: "blue" },
    { label: "C", percentage: 20, color: "green" }
]);
pieChart.set("showLabels", true);
pieChart.set("labelFontSize", 14);
pieChart.set("labelColor", "black");

canvas.draw();

Preview

Available Methods

  • set("radius", 100) → Set the radius of the pie chart.
  • set("data", [...]) → Set the data for the pie chart (array of `{ label, percentage, color }`).
  • set("showLabels", true) → Show or hide labels.
  • set("labelFontSize", 14) → Set the font size for labels.
  • set("labelColor", "black") → Set the color of the labels.
  • set("globalAlpha", 1) → Set transparency (1 = fully visible, 0 = fully hidden).

Available Getters/Setters

  • x → Get or set the X position.
  • y → Get or set the Y position.

Bounding Box Methods

  • boundingRectangleX() → Get the X position of the bounding box.
  • boundingRectangleY() → Get the Y position of the bounding box.
  • boundingRectangleWidth() → Get the width of the bounding box.
  • boundingRectangleHeight() → Get the height of the bounding box.

Works Without Assets

The Pie Chart Item does not require taleem-assets. You can use it directly with TaleemCanvas without any external dependencies.
